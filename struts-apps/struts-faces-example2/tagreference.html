Tag reference sheet

Tag library reference for the following tag libraries:

  • Application Tag Library (app.tld) - uri:http://jakarta.apache.org/taglibs/struts-example-1.0

Application Tag Library - Version 1.0

This tag library contains functionality specific to the Struts Example Application.

Namespace definition: xmlns:app="http://jakarta.apache.org/taglibs/struts-example-1.0"

Tags

  • checkLogon Validate that there is a currently logged on user, by checking for the existence of a session-scope bean under the specified name.
  • linkSubscription Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified Subscription object.
  • linkUser Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified User object.

Required attributes are marked with a *

<app:checkLogon>

Validate that there is a currently logged on user, by checking for the existence of a session-scope bean under the specified name. If there is no such bean, forward control to the specified page, which will typically be a logon form.

Can contain: empty

Attributes

Name Description Type
name
Name of the session-scope bean to check for [user]
String
page
Context-relative path to the logon page [/logon.jsp]
String

<app:linkSubscription>

Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified Subscription object.

Can contain: JSP

Attributes

Name Description Type
name
Name of the attribute containing the specified Subscription. [subscription]
javax.el.ValueExpression
page*
Context-relative URL of the generated link
javax.el.ValueExpression

<app:linkUser>

Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified User object.

Can contain: JSP

Attributes

Name Description Type
name
Name of the attribute containing the specified user. [user]
String
page*
Context-relative URL of the generated link
String
