Logic Taglib Guide

The Logic Taglib

Note: - Many of the features in this taglib are also available in the JavaServer Pages Standard Tag Library (JSTL). The Apache Struts group encourages the use of the standard tags over the Struts specific tags when possible.

The "struts-logic" tag library contains tags that are useful in managing conditional generation of output text, looping over object collections for repetitive generation of output text, and application flow management
